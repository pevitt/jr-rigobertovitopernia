# Rigoberto Vito Pernía — Memoria de Seboruco

Página web personal y memorial del primer alcalde del Municipio Autónomo Seboruco, Táchira, Venezuela.

## Estructura

```
index.html          → Página principal
styles.css          → Estilos
image-slot.js       → Componente drag & drop para fotos
assets/             → Imágenes (retrato, portadas de libros, fotos del pueblo)
```

## Secciones

- **Sobre Él** — Biografía completa y sus propias palabras
- **Seboruco** — Historia, línea de tiempo, carrusel de fotos y mapa
- **Libros** — Descarga de sus libros en PDF
- **Contacto** — Formulario y correo de contacto

## Cómo actualizar

1. Edita los archivos localmente
2. Haz `git add .` → `git commit -m "descripción del cambio"` → `git push`
3. Netlify despliega automáticamente en segundos
